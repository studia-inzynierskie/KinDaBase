# KinDaBaseTurboProject



## Opis Projektu

Projekt realizowany w ramach przedmiotu BD1 (Bazy Danych 1) na studiach.

Baza została zaludniona i przetestowana skryptami SQL, a także prostą aplikacją Pythonową.
